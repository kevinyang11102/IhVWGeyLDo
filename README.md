# PHP+Mysql游戏购物信息管理系统 java872

## 项目概述

本项目是基于PHP和Mysql开发的游戏购物信息管理系统，分为前台展示和后台管理两部分。系统提供登录、注册、购物车、订单、留言等基础功能，支持游戏信息的增删改查和图片上传。

## 技术栈

- 后端语言：PHP
- 数据库：Mysql
- 集成环境：phpstudy（推荐）、Wampsever、Xampp

## 功能模块

### 前台功能

- 用户注册与登录
- 游戏信息浏览与分类
- 游戏详情页与收藏
- 购物车管理与订单结算
- 个人中心信息修改与头像上传

### 后台功能

- 用户管理（删除操作）
- 游戏信息管理
- 发布新游戏信息
- 后台首页游戏数量占比展示

## 系统角色

- 普通用户：浏览游戏、收藏游戏、添加购物车、生成订单、管理个人资料。
- 管理员：管理用户、发布及管理游戏信息、查看游戏数据统计。

## 运行环境

- PHP版本在5.6以上

## 目录结构

```
/docs              // 文档目录
/includes          // 包含文件目录
/libraries         // 类库目录
/models            // 模型目录
/views             // 视图目录
/controllers       // 控制器目录
/config            // 配置文件目录
/assets            // 资源目录（如CSS、JS、图片等）
```

## 部署步骤

1. 准备PHP和Mysql的运行环境。
2. 将项目文件部署到服务器。
3. 导入数据库文件到Mysql数据库。
4. 根据实际情况修改配置文件。
5. 访问前台或后台进行系统使用。
